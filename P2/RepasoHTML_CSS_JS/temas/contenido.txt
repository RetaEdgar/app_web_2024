HTML (hyperText Markup Language) es el lenguaje estandar utilizado para crear y estructurar páginas web.
No es un lenguaje de programación, sino un lenguaje de marcado, lo que significa que su principal función es
organizar y definir el contenido de una página web mediante etiquetas.
